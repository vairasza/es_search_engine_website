Guide on using the Search Engine:

  • The Search Engine can handle multiple input parameters at once.

  • If you want to include a parameter e.g. title click the checkbox next to the title.

  • Checkboxes can be "must" (input field context must be contained) or "should" (input field context can be included). See their title if you hover over the checkboxes.

  • In the title and content input field, you can type in multiple terms seperated by command followed by whistespace.

    • If you want to exlcude a term in title or content, start the term with a "!".

    • Start with "?" as an additional operator if the term should be included in the search engines answer.

    • example: "hillary clinton, !donald trump, ?white house, !america, USA"

  • published from ... to ... requires a valid date format.
